## Tipos de planos

### internal

Plano exclusivo para usuários do tipo `internal`.

Possui todos os módulos e recursos disponíveis no sistema.

### starter

Plano inicial para empresas que estão começando a utilizar o sistema.

Inclui os módulos:

- Dashboard
- Clientes
- Pets
- Agendamentos
- Serviços
- Usuários
- Configurações

### basic

Plano intermediário para empresas que já utilizam o sistema há algum tempo e/ou necessitam de mais funcionalidades.

Inclui os módulos:

- Todos os módulos do plano [starter](#starter)
- Módulo de Planos de Serviços
- Módulo de Relatórios

### essential

Plano avançado para empresas que já utilizam o sistema há algum tempo.

Inclui os módulos:

- Todos os módulos do plano [basic](#basic)
- Módulo de Produtos
- Módulo de Agendamentos por Profissionais
- Módulo de Tele-busca/Entrega de Pets
- Módulo de Estoque

### premium

Plano completo para empresas que já utilizam o sistema há algum tempo.

Inclui os módulos:

- Todos os módulos do plano [essential](#essential)
- Módulo de Relatórios Personalizados
- Módulo de Creche de Pets
- Módulo de Hotel de Pets
- Módulo de Chat
- Módulo de Notificações
- Módulo Financeiro
- Módulo de Fornecedores
- Módulo de Acesso de Usuários Externos

### trial

Plano de demonstração para empresas que não utilizam ou que estão começando a utilizar o sistema.

Inclui todos módulos com dados de demonstração.

O plano trial possui as seguintes características:

- Duração de 7 dias (para usuários `free`)
- Não pode ser renovado
- Não pode ser convertido em outro plano
- Não pode ser cancelado
- Não pode ser reativado
- Não pode ser desativado

- Não pode ser copiado
